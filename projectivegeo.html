\section{The Axioms} 
 
	\begin{definition} 
		A \emph{projective plane} is a triple $\mathcal{(P, L, I)}$. The set $P$ consists of the points, and the set $L$ consists of the lines of the geometry. The inclusion $I \subseteq P \times L$ is an incidence relation satisfying the following three axioms: 
			\be 
				\item[i)] For any two distinct points, there is exactly one line incident with both of them. 
				\item[ii)] For any two distinct lines, there is exactly one point incident with both of them.  
				\item[iii)] There are four distinct points such that no line is incident with more than 
				two of them. 
			\ee 
	\end{definition} 
 
	\begin{remark} 
		The first two axioms describe a completely symmetric relation of points and lines. The second axiom simply states that (without any exception) two distinct lines will always intersect in a unique point. The first axiom states that (without any exception) two distinct points will always have a line joining them. The third axiom merely ensures that the structure is not a degenerate trivial case in which most of the points are collinear 
	\end{remark} 
 
	There are various models of the projective spaces. We will build the first one (the most popular one) based on the Euclidean space. Let $\mathcal{(P_{\mathbb E}, L_{\mathbb E}, I_{\mathbb E})}$ be set of points, lines and the usual incidence relation in Euclidean space respectively. For each line $l$ in $\R^n$ we will associate the equivalent class of lines parallel to $l$ with a point $p_{[l]}$ and consider this in the intersection of these lines (points at infinity). And for every point at infinity, we associate the line $l_\infty$ where all $p_{[l]}$ lies on (line at infinity). Then we will have the first model of projective space as follow:  
		\be 
			\item $\mathcal P = P_{\mathbb E} \cup p_{[l]}$ 
			\item $\mathcal L = L_{\mathbb E} \cup l_\infty$ 
			\item $\mathcal P = P_{\mathbb E} \cup p_{[l]}$ 
		\ee 
 
\section{Some Small Examples} 
	 
 
	Let's start with the smallest example of a projective plane. Since the plane above contain infinitely many points, a question can be asked is that is there a projective plane with finitely many points and lines? An example would be  Fano plane. Axiom (iii) tells us that it must contain at least four points, no three of which are collinear. Let the 
    four points be $A, B, C,$ and $D$. Then every line through each of these points must intersect each other. There are exactly three missing intersections, namely those of the pairs of lines $(AB, CD), (AC, BD),$ and $(AD, BC)$ 
    \begin{center} 
    	\includegraphics[scale=0.5]{Fano plane} 
    \end{center} 
     
    Using the axioms, we can prove the following lemmas:  
    \begin{lemma} 
    	Given $p, q \in \cal P$ and $l,m \in \cal L$ such that $pIm, qIm, pIl, qIl$ for $I$ is the incident relation, then either $p=q$ or $l=m$ 
    \end{lemma} 
 
    \begin{lemma} 
    	Every line in a projective plane incident with at least 3 distinct points 
    \end{lemma} 
 
    \begin{lemma} 
    	For every point $p$ there exists at least 1 line not incident with $p$ 
    \end{lemma} 
 
    And a nice theorem about the number of points and lines in a finite projective plane:  
    \begin{theorem} 
    	Let $\mathcal(P, L, I)$ be a projective plane with finite sets $\cal P$ and $\cal L$. Then there exists a number $n \in \N$ such that $|p(l)| = n + 1$ for any $l \in  \cal L$ and $|l(p)| = n + 1$ for any $p \in \cal P$. 
    \end{theorem} 
    \begin{proof} 
    	 
    	Idea: first we need to show that given 2 distinct lines $m$ and $l$ that passing through the same point $p$ and that there are $n$ points other that $p$ on $l$ then there must also be $n$ point like that on $m$. This can be show by look at the join of a point $q$ not on $l$ with the other $n$ points on $l$.\\ 
    	Then look at the direction where $p$ is not on $l$ and there are $n+1$ lines generated by joining $p$ with $n+1$ points on $l$ 
 
    \end{proof} 
 
    The number $n$ of the last proposition (which was the number of points on a line minus one) is usually called the order of the projective plane. The following proposition relates the order and the overall number of points and lines in a finite projective plane. 
 
    \begin{theorem} Let $\mathcal(P, L, I)$ be a projective plane with finite sets $\cal P$ and $\cal L$ of order $n$. Then we have $|P| = |L| = n^2 + n + 1$. 
     
    \end{theorem} 
 
   	\begin{proof}(Sketch:) Think of $(n+1)n+1=n^2+n+1$ 
   	\end{proof}

\subsection{A Spatial Point of View} 
	Let $\K$ be any arbitrary field. We define the following equivalence class on $\K^3$:  
	$$[v]:= \{v'\in \K^3|\lambda v=v', \lambda \in \K- \{0\}\}$$ 
	The set of all such equivalence class will be denoted by $\ds \frac{\K^3 -\{0,0,0\}}{\K -\{0\}}$. We define the orthogonal relation for the equivalence class in the canonical way that $\cyclic{p,l}=0 \LRA [p] \perp [l]$. So in away we think of vector and point interchangably and with this current setting a point is an equivalence class. \\ 
 
	We set ${\cal{P_\K}} =\ds \frac{\K^3 -\{0,0,0\}}{\K -\{0\}}$  and ${\cal{L_\K}} =\ds \frac{\K^3 -\{0,0,0\}}{\K -\{0\}}$ (consider $\cal P_\K$ and $\cal L_{\K}$ as 2 disjoint copies of the same space). We define the incident relation on these 2 sets as for $[p] \in \cal P$ and $[l] \in \cal L$ that $p I l$ if $[p] \perp [l] $  
 
	\begin{claim} 
		This is indeed a projective plane 
	\end{claim} 
	 
	\begin{proof} 
		For axiom 1, note that we need to show that 2 distinct points can only have 1 line incident between them. Since $p$ and $q$ are distinct they cannot just be different by a constant multiple. Then, let $p=(p_1,p_2,p_3)$ and $q=(q_1,q_2,q_3)$ be a representative of this equivalence class and put them into a $[2x3]$ matrix where each of the vector is a row. Then this matrix is of rank 2 and so its kernel is of rank 1 which is 1 dimensional. But this is just simpy the claim since every line in the same 1 dimensional vector space only differ by a scalar thus is the same point in this projective plane. \\ 
 
		For axiom 2, do the similar argument with $\cal L$\\ 
 
		For axiom 3, abusing the notation $0$ and $1$ as the zero and one of the field. we have the colinearity of tehse 4 points $(0,0,1), (1,1,1), (0,1,1), (1,0,1)$ 
	\end{proof}
